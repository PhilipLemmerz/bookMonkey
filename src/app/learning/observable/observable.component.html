Observables

Create Functions

of erstellt ein Observable aus einzelnen Werten: const $obs = of(1,2,3)

from erstellt eines aus einem array const $obs = from(myArray)

timer(500) -> erzeugt alle 500ms einen neuen Wert // ---1---2---3---4---5....

timer(0,500) -> Startet bei 0ms und erzeugt den Wert und dann alle 500ms // 1---2---3---4---5...
